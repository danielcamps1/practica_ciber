Arquitectura de la Red Neuronal

Definición del modelo secuencial y configuración de capas:

model = Sequential([
  Conv2D(32, (3, 3), activation='relu', input_shape=(32, 32, 3)),
  BatchNormalization(),
  MaxPooling2D((2, 2)),
  Conv2D(64, (3, 3), activation='relu'),
  BatchNormalization(),
  MaxPooling2D((2, 2)),
  Dropout(0.2),
  Flatten(),
  Dense(128, activation='relu'),
  Dense(10, activation='softmax')
])
La arquitectura utiliza capas Conv2D para la extracción de características, BatchNormalization para la estabilidad y Dropout para mitigar el sobreajuste, logrando un 74.40% de precisión inicial.

Defensa: Fine-Tuning Adversarial

Se ha aplicado una técnica de Fine-Tuning sobre el modelo original utilizando ejemplos perturbados generados dinámicamente.

Entrenamiento con Defensa: El modelo ajusta sus pesos internos para minimizar la pérdida tanto en imágenes limpias como en muestras con ruido FGSM.

Comparativa de Robustez en Test

Esta comparativa evalúa la precisión del conjunto de test bajo diferentes niveles de estrés. Como se observa, el ataque afecta a ambos modelos, pero la degradación es significativamente más lenta en el modelo defendido.

Conclusiones Técnicas

1. Los ataques adversarios modernos pueden vulnerar modelos de alta precisión de forma casi invisible.

2. El Fine-Tuning con ejemplos adversarios mejora la resiliencia del sistema.

3. Observación: Ningún modelo es invulnerable; la defensa busca elevar el coste del ataque y mantener la operatividad el mayor tiempo posible.
